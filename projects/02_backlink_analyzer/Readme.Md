## ğŸš¨ **Troubleshooting**  
- **Error: "Moz API Limit Reached"**  
  The free tier has rate limits (2s between requests). Upgrade for higher limits.  
- **No backlinks found**  
  Try different search operators like `link:example.com -site:example.com`
